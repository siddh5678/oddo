Technical Team Assignment Guide

Complete guide on how teams are assigned to equipment and maintenance requests

Assignment Flow
Equipment Creation

When creating or editing equipment, select a Maintenance Team from the dropdown.

Location: Equipment → Create/Edit → Maintenance Team field
Automatic Team Assignment

When creating a maintenance request and selecting equipment, the system automatically assigns the team from that equipment.

No manual selection needed - happens automatically!
Technician Selection

The technician dropdown shows only technicians from the assigned team. Select a technician from the filtered list.

System validates that technician belongs to the team
Request Created

Maintenance request is created with team and technician assigned. Ready for workflow!

Key Features
Auto-Assignment

Team automatically assigned from equipment - no manual work needed

Validation

System ensures technician belongs to assigned team

Manual Override

Can manually change team in request if needed

Filtered Technicians

Only team members shown in technician dropdown

Quick Steps
  1. Create/Edit Equipment
  2. Select Maintenance Team
  3. Save Equipment
  4. Create Maintenance Request
  5. Select Equipment (Team auto-assigned)
  6. Select Technician (from team)
  7. Complete Request
Important Rules
  • Equipment must have a team assigned
  • Technician must be team member
  • Team can be changed manually in request
  • Multiple technicians per team allowed
